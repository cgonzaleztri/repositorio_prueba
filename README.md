# repositorio_prueba
Este es un repositorio de prueba


1. Crear repositorio
2. Añadir colaborador desde github web
3. Para hacer cambios desde R:
abrir archivo
modificar
hacer un commit (si o si, para mantener el control de cambios)
hacer un PULL (subirlo)
y la otra persona ya puede descargarlo haciendo un PUSH

Si eres el colaborador:
coger link del propietario y crear proyecto en su propio R con el link
